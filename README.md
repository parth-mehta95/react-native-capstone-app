# React Native Capstone App

## Project Overview

This is a React Native project created using Expo. The project contains reusable ES6 utility modules, JSDoc comments, and a simple example of async/await API usage.

## Features

- React Native app created with Expo
- Clean project structure
- ES6 utility modules
- JSDoc comments for all utility functions
- Async/await API helper example
- Date formatting utilities
- Form validation utilities

## Folder Structure

## Utility Modules

### apiHelper.js

This file contains API-related helper functions.

Functions:

- buildQueryString
- fetchPosts
- createPost

### dateFormatter.js

This file contains date-related helper functions.

Functions:

- formatDate
- addDays
- getCurrentYear

### validators.js

This file contains form validation helper functions.

Functions:

- isRequired
- isValidEmail
- isValidPassword
- validateUserForm

## Local Setup Guide

### Prerequisites

Install the following:

- Node.js
- npm
- Git
- Expo Go mobile app

### Installation Steps

1. Clone the repository:

git clone https://github.com/your-username/react-native-capstone-app.git

2. Go into the project folder:

cd react-native-capstone-app

3. Install dependencies:

npm install

4. Start the project:

npm start

5. Run the app:

- Scan the QR code using Expo Go
- Press `a` to run on Android emulator
- Press `i` to run on iOS simulator
